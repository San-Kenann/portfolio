01 / BTS SIO · Développement Web
🌍 Site de réservation de voyages
Conception et développement d'un site web complet de réservation de voyages en ligne. Le projet comprend une interface utilisateur en HTML/CSS permettant la recherche et la réservation de destinations, un back-end en PHP gérant la logique métier (connexion, panier, confirmation), ainsi qu'une base de données MySQL pour stocker les utilisateurs, les voyages disponibles et les réservations. Mise en place d'un système d'authentification sécurisé (sessions PHP) et d'un espace administrateur pour gérer le catalogue.
Fonctionnalités
Recherche de voyages Système de réservation Authentification Espace admin
HTML CSS PHP MySQL Sessions PHP CRUD

02 / BTS SIO · Infrastructure Réseau

🏫 Infrastructure réseau — Lycée Robert Schuman

Simulation physique complète · Dugny (93)

Contexte du projet

Le lycée privé Robert Schuman (Dugny, 93) est un établissement sous contrat d'association avec l'État, fondé en 1920. Il accueille environ 349 élèves et 30 professeurs. Ce projet consiste à simuler et documenter l'intégralité de son infrastructure réseau dans le cadre du BTS SIO SISR.

Architecture serveurs
🖥 Réseau ADMINISTRATION
• Windows Server 2008 R2
📚 Réseau PÉDAGOGIE
• Windows Server 2008 R2 (Pédagogie)
• Windows Server 2012 R2 (Exchange)
• Windows Server 2008 (TSE)
• Proxy Linux
📡 Accès Internet : Orange ADSL 8 Mo + SDSL 2 Mo  ·  Nom de domaine : lprs.org
